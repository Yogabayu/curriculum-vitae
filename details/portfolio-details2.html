  • Kategori: Alat Pengukur Cuaca Otomatis
  • Tanggal: 01-08-2020
  • Project URL: www.cuacaku.xyz

Alat Pengukur cuaca otomatis ini memiliki 5 Sensor dengan 6 parameter, 5 Sensor tersebut adalah Sensor Curah Hujan, Sensor Kecepatan Angin, Sensor Suhu + Kelembaban Udara, Sensor Ultrasonik, dan Sensor Suhu Permukaan air. Sedangkan 6 Parameter adalah curah hujan, kecepatan angin, suhu udara, kelembaban udara, penguapan air, dan suhu permukaan air. Semua data sensor akan dikirim ke database dan ditampilkan dalam website dan aplikasi android. Alat ini dapat sepenuhnya beroperasi tanpa menggunakan listrik dari PLN karena mempunyai Panel surya 50wp sebagai sumber daya nya. Serta dilengkapi data logger untuk mencegah kehilangan data jika terjadi sesuatu pada alat.
