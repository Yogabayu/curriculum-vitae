Saya tergabung dalam tim SIMRS untuk membangun web ini. Dengan fokus pekerjaan saya pada UI dan UX dari website RSUD Kertosono ini. Website ini merupakan website berbasis CodeIgniter 3 yang memiliki fasilitas admin dan author untuk mengelola website ini. Dalam mendesign web ini saya berprinsip Simple Bermanfaat. Kenapa saya menggunakan prinsip tersebut karena web pelayanan umum harus mudah diakses dengan tampilan yang tidak terlalu berat saat loading-nya, serta dapat dengan mudah memberi informasi kepada pengunjung situs.
